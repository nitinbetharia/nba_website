JavaScript Required

To access all features of our website, please enable JavaScript in your browser.

Our website uses JavaScript to provide enhanced functionality and interactive features. While you can still browse basic content, many features will not work properly without JavaScript enabled.

Features that require JavaScript:

  • Tax Calculator
  • SIP Calculator
  • EMI Calculator
  • PPF Calculator
  • NPS Calculator
  • Contact Form
  • Newsletter Subscription
  • Interactive Navigation
  • Smooth Scrolling

How to Enable JavaScript

Google Chrome

1. Click the three dots menu → Settings → Privacy and security → Site Settings → JavaScript → Allow

Mozilla Firefox

1. Type "about:config" in address bar → Search for "javascript.enabled" → Set to true

Safari

1. Safari menu → Preferences → Security → Enable JavaScript checkbox

Microsoft Edge

1. Three dots menu → Settings → Site permissions → JavaScript → Allow
